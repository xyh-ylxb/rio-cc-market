# Rio Agent Repo Research Analyst

Use this skill when you need fast orientation in a repository.

## Workflow

1. Read key project docs.
2. Map the relevant subdirectories and architecture boundaries.
3. Search for similar implementations and naming patterns.
4. Note contribution and testing conventions that affect the task.

## Output

- key files
- patterns to follow
- conventions to preserve
