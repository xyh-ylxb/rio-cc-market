# Rio Compound Docs

Use this skill after a problem has been solved and verified.

## Workflow

1. Pick an appropriate documentation location in the repository.
2. Record:
   - module
   - date
   - problem type
   - symptom
   - root cause
   - fix
   - verification
3. Keep one problem per document.
4. Use the bundled schema as a checklist.
